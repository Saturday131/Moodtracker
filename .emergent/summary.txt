The user wants to build a full-stack mobile mood tracker app using Expo.

**Initial MVP Requirements:**
- Record daily moods with optional notes.
- Use emojis for different moods.
- View mood history in a calendar or graph view.
- Export mood data.

**Feature Enhancements Added:**
1.  **Advanced Tracking:** Allow three mood entries per day (Morning, Midday, Evening) with a multi-layered scoring system (e.g., Overall, Energy, Calm). Provide advanced analytics based on this data.
2.  **AI Chatbot:** Implement a chatbot (MoodBuddy) that can analyze mood trends, answer user questions about their data, and provide a weekly summary via push notification.
3.  **Multimedia Notes:** Add a feature to create notes in text, voice, or image format. The chatbot should analyze these notes, provide summaries, and use them to give more contextual reminders and insights.
4.  **Intelligent Reminders:** Schedule reminders intelligently based on the content of the notes (both text and voice). These reminders should be sent via push notifications. The app should also feature a browsable library of all past notes.
